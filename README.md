# PRODINTEC Energy Analyzer

Software profesional para el análisis técnico y financiero de sistemas de generación solar fotovoltaica.

## Objetivos

- Dimensionamiento de sistemas fotovoltaicos.
- Evaluación financiera.
- Generación de reportes.
- Gestión de proyectos.
- Certificación UPME.
- Estudios RETIE.

## Tecnologías

- Python 3.13
- PySide6
- SQLite
- Git
